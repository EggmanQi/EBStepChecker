# EBStepChecker
iOS steps checker

* iOS平台的计步器，数据结果来源于设备的重力加速计
* 可以在不使用芯片（iOS7，iPhone5S 之后机型）的计步数据情况下获得步数
* 内置了转换为使用芯片计步的功能
* 最新的代码在“数据持久化分支”
